- Two or more real Claude panes spawned via `POST /api/sessions` with
  `{"cli":"cmd.exe","args":["/c","claude"],"cwd":"test-workspace/pane-[a|b|c]"}`.

  synthetic-event endpoint is available for edge cases.
- Every claim of "green" requires a paste-able artefact: a backend response
  body, a file written by Claude, or a screenshot.

## E. How to re-run a flow

```bash
# boot a fresh dogfood backend on :4800 (kills anything on that port first)
cd "G:/_OneDrive/OneDrive/Desktop/Py Apps/wezbridge"
pkill -f 'tsx.*theorchestra-start'
rm -rf test-workspace/_sessions test-workspace/token.json
THEORCHESTRA_PORT=4800 \
  THEORCHESTRA_TOKEN_FILE="$(pwd)/test-workspace/token.json" \
  THEORCHESTRA_SESSIONS_DIR="$(pwd)/test-workspace/_sessions" \
  THEORCHESTRA_DECISIONS_DIR="$(pwd)/test-workspace/_decisions" \
  THEORCHESTRA_CONFIG_FILE="$(pwd)/test-workspace/orch-cfg.md" \
  THEORCHESTRA_TASKS_FILE="$(pwd)/test-workspace/active_tasks.md" \
  node node_modules/tsx/dist/cli.mjs scripts/theorchestra-start.ts \
  > test-workspace/boot.log 2>&1 &

# token
TOKEN=$(python -c "import sys,json;print(json.load(open('test-workspace/token.json'))['token'])")

# spawn two Claude panes
curl -s -X POST -H "Authorization: Bearer $TOKEN" -H "Content-Type: application/json" \
  -d '{"cli":"cmd.exe","args":["/c","claude"],"cwd":"…/test-workspace/pane-a","tabTitle":"pane-a"}' \
  http://127.0.0.1:4800/api/sessions
curl -s -X POST -H "Authorization: Bearer $TOKEN" -H "Content-Type: application/json" \
  -d '{"cli":"cmd.exe","args":["/c","claude"],"cwd":"…/test-workspace/pane-b","tabTitle":"pane-b"}' \
  http://127.0.0.1:4800/api/sessions

# …then run the flow under test (A2A handoff / auto-handoff / queue / inject) …

# kill the panes when done
for sid in $(curl -s -H "Authorization: Bearer $TOKEN" http://127.0.0.1:4800/api/sessions \
  | python -c "import sys,json;[print(s['sessionId']) for s in json.load(sys.stdin)]"); do
  curl -s -X DELETE -H "Authorization: Bearer $TOKEN" \
    http://127.0.0.1:4800/api/sessions/$sid >/dev/null
done
```
